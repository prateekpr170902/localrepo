ğŸ”¹ RESOLVING MISTAKES / RESET
Case 1: Go one commit back

ğŸ‘‰ Removes last commit but keeps code.

Case 2: Remove last commit completely

ğŸ‘‰ Removes commit and code (dangerous).

Case 3: Reset to specific commit

ğŸ‘‰ Moves HEAD to that commit.

Force reset (dangerous)

ğŸ‘‰ Everything after that commit is deleted.
